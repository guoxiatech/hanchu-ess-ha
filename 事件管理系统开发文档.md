## 事件解析逻辑

### 1. 单值匹配（Single Value）

**场景：** 信号量P058，字段值等于某个值时触发事件

**配置示例：**
```json
{
  "signal_code": "P058",
  "parse_type": "single",
  "field_code": "status",
  "match_value": "5",
  "event_code": "E001",
  "event_name": "设备故障"
}
```

**解析逻辑：**
```java
if (fieldValue.equals(matchValue)) {
    triggerEvent(eventCode, eventName);
}
```

### 2. 二进制位匹配（Binary Bit）

**场景：** 信号量P066，解析二进制数组特定位置

**配置示例：**
```json
{
  "signal_code": "P066",
  "parse_type": "binary",
  "array_index": 3,
  "array_length": 16,
  "event_code": "E002",
  "event_name": "过压告警"
}
```

**解析逻辑：**
```java
public static int getBit(int num, int i) {
    if ((num & (1 << i)) != 0) {
        return 1;
    } else {
        return 0;
    }
}

// 使用
if (getBit(value, arrayIndex) == 1) {
    triggerEvent(eventCode, eventName);
}
```

---

### 3. 数组多值匹配（Array Enum）

**场景：** 数组值匹配枚举列表

**配置示例：**
```json
{
  "signal_code": "P070",
  "parse_type": "array",
  "match_value": "[1,3,5]",
  "event_code": "E003",
  "event_name": "多重故障"
}
```

**解析逻辑：**
```java
// 转换为List
String arrayStr = value.toString().replaceAll("[\\[\\]]", "");
List<String> valueList = Arrays.asList(arrayStr.split(","));
List<String> matchList = Arrays.asList(matchValue.replaceAll("[\\[\\]]", "").split(","));

// 检查是否包含匹配值
for (String match : matchList) {
    if (valueList.contains(match.trim())) {
        triggerEvent(eventCode, eventName);
        break;
    }
}
```

### 4. 电池事件解析（BMS）

**场景：** BMS协议字段统一处理

**解析逻辑：**
```java
public static Set<String> calEventCodes(IotBmsDailyUpload iotBmsDailyUpload) {
    Set<String> set = new HashSet<>();
    for (BmsProtocolField.GROUP group : BmsProtocolField.GROUP.values()) {
        if (group == BmsProtocolField.GROUP.ALARM || 
            group == BmsProtocolField.GROUP.PROTECT || 
            group == BmsProtocolField.GROUP.FAULT) {
            Map<String, Object> map = iotBmsDailyUpload.getColumnMap(group);
            map.forEach((k, v) -> {
                if (ObjectUtils.isNotNull(v) && "1".equals(v.toString())) {
                    set.add(k);
                }
            });
        }
    }
    return set;
}
```

#### 触发逻辑

```java
public void triggerPushNotification(DeviceEvent event) {
    // 1. 根据事件编码查询推送规则
    EventPushRule rule = eventPushRuleService.findByEventCode(event.getEventCode());
    
    if (rule == null || !rule.isEnabled()) {
        return;
    }
    
    // 2. 获取消息模板
    MessageTemplate template = messageTemplateService.getById(rule.getMessageTemplateId());
    
    // 3. 渲染消息
    String title = renderMessage(template.getI18nTitleKey(), event);
    String content = renderMessage(template.getI18nContentKey(), event);
    
    // 4. 推送给目标角色
    for (String role : rule.getTargetRoles()) {
        for (String channel : rule.getPushChannels()) {
            pushService.send(role, channel, title, content, event);
        }
    }
}
```

### 3. 设备健康度分析

#### 健康度计算规则

```java
// 健康度 = 100 - (事件权重总和)
// 权重：critical=20, error=10, warning=5, info=1

int healthScore = 100;
healthScore -= criticalCount * 20;
healthScore -= errorCount * 10;
healthScore -= warningCount * 5;
healthScore -= infoCount * 1;
return Math.max(0, healthScore);
```

### 4. 解决建议获取逻辑

```java
public String getSolutionByRole(String eventCode, String roleType, JSONObject solutionConfig, String defaultKey) {
    // 1. 尝试从 JSON 配置获取角色特定解决建议
    if (solutionConfig != null && solutionConfig.containsKey(roleType)) {
        String roleKey = solutionConfig.getString(roleType);
        if (StringUtils.isNotBlank(roleKey)) {
            return i18nService.getMessage(roleKey);
        }
    }
    
    // 2. 使用默认解决建议
    return i18nService.getMessage(defaultKey);
}
```
